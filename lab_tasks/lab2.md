# Лабораторная работа 2. Выполнение - индивидуальное

В данной лабораторной работе вам необходимо реализовать 2 класса: `stack` и `queue` на стеках. Для имплементации класса `stack` вам необходимо использовать класс `DynamicArray`, который вы делали при сдаче домашней работы. Класс `DynamicArray` может быть дополнен относительно имплементации в ДЗ.  

## Требования:
  - Для класса stack необходимо имплементировать:
    - `push(x)` - добавить элемент в стек
    - `top()` - посмотреть на элемент, находящийся на вершине стека без удаления
    - `pop()` - удалить элемент на вершине стека и вернуть его
    - `isEmpty()` - проверка стека на пустоту
  - Класс `stack` обязан быть шаблонным и использовать внутри класс `DynamicArray`
  - Класс `stack` имеет фиксированный максимальный размер, переполнение ведет в выбросу исключения `std::overflow_error`.
  - Класс `queue` должен быть шаблонным и имплементирован с использованием класса stack
  - Для класса queue необходимо имлементировать:
    - `enqueue()` - добавление элемента в очередь
    - `dequeue()` - удаление элемента из очереди и его возвращение
    - `isEmpty()` - проверка очереди на пустоту
    - `size()` - возвращает кол-во элементов в очереди
